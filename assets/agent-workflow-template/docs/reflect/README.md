# Reflections

This directory is only for severe recurring bug lessons.

Add a reflection when the issue is all of:

- Hard to locate.
- High impact.
- Likely to recur.
- Relevant to production stability, correctness, data integrity, security, privacy, cost, or user trust.

Do not add routine instruction feedback, naming preferences, ordinary review comments, or one-off reminders here. Put those in `AGENTS.md`, `docs/agent/`, `docs/patterns/`, module docs, or changelogs.

Recommended structure:

```markdown
# YYYY-MM-DD - Bug Name

## Summary

## Impact

## Root Cause

## Detection

## Fix

## Prevention Rules

## Related Tests

## Related Docs
```
